## (kbm) Kanovel Components Style order

```js
// position components
pos(),
z(),
fixed(),
follow(),

// transform components
anchor(),
scale(),
rotate(),

// primary render components (is the main render)
sprite(),
text(),
rect(),
circle(),
uvquad(),

// secondary render components (affect the main render)
color(),
opacity(),
shader(),
outline(),

// physics related components
area(),
body(),
doubleJump(),

// behaviour and functions components
stay(),
lifespan(),
offscreen(),
move(),
health(),
timer(),
fadeIn(),
```
