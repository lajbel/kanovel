## Quick Example

```js
import kanovel from "kaboom";

kanovel();

character("b", "Beany", "beany");
character("m", "Marky", "marky");

chapter("start", () => [
    say("b", "He is waiting"),

    show("m", "center"),
    say("m", "Who?"),

    say("b", "What?"),
]);
```
